"#url_shortner_project"
A clean  and simple URL shortener built with django and react ,paste  a long URL and get a short link instantly.

   **Features
- ** url shortening :** Convert long URLs to short codes
- ** Fast Redirects : **Click short link to  go to original URL
- ** SQLite Database : ** Stores all URLs locally
- ** Admin Panel : ** Django admin  to view all links
    **Tech Stack
-**Backend :** Django + Django REST Framework
-**Database : ** SQLite
-**Frontend : ** React - [url_shortner_project_frontend](https://github.com/Aishwarya980/url_shortner_project_frontend.git)
